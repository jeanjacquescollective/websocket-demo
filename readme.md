
# WebSocket Demo

A simple demonstration of WebSocket communication.

## Overview

This project showcases WebSocket implementation for real-time bidirectional communication between client and server.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running the Project

```bash
npm start
```

## Usage

Connect to the WebSocket server and send/receive messages in real-time.

## Features

- Real-time communication
- Bidirectional data transfer
- Simple client-server architecture
